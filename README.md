# Go Do Stuff — Berlin

A free, mobile-first guide to 160 hidden, historically rich places in Berlin for young people aged 12-25.

## Features
- 160 curated places with stories, "imagine" time-travel sections, and transit directions
- 22 self-guided missions
- 18 themed collections
- Trip planner with area-based routing
- Risk labels and safety warnings on every place
- Works offline (PWA)

## Install on your phone
1. Open the website in your phone's browser
2. **iOS**: Tap Share > "Add to Home Screen"
3. **Android**: Tap the three dots menu > "Add to Home Screen" or "Install App"

## Host on GitHub Pages
1. Push this folder to a GitHub repo
2. Go to Settings > Pages > Source: Deploy from branch (main)
3. Your app will be live at `https://yourusername.github.io/godostuff/`

## Tech
- Single-file React app (no build step)
- Babel transpiles JSX in-browser
- Tailwind CSS from CDN
- Service worker for offline caching
- No backend, no database, no tracking
